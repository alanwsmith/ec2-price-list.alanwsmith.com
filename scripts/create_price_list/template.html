AWS EC2 Instance Pricing

Trying to figure out EC2 pricing never fails to frustrate me. This page isn't bad, but I wanted things sorted by price within each instance category. So, I banged my head against the price list API for a while and built it.

The first table is GPU instances sorted by card type. The second table includes all instance types with GPU instances sorted directly by price (regardless of card type). In each case, the price is for On Demand Linux instances with no addiitonal software installed.

Prices are from Sept. 29, 2022. The source code is here if you'd like to get current prices.

GPU Instances By Card Type

$GPU_TABLE_ROWS
GPU Card Name vCPUs GPUs Memory $/Hour

All Instance Types

$TABLE_ROWS
Family Name vCPUs GPUs Memory $/Hour

Notes

  • Prices are in USD
  • I put this togehter because I'm working on a comparison of EC2 instances for rendering blender files. I couldn't get my head around the pricing from the AWS UI and since that's always bugged me we ended up here
  • I didn't put in any of the storage or network notes. There's a bunch of different types that would have added clutter to this basic price list that I was after
  • If you'd like to dig into the pricing API to make your own views I've got a good starting point setup here
  • As mentioned above, this data doesn't update live. If you want the current values you can grab the source code here
